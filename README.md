# nakrut

Small script for getting viewers on targeted site. 

## Usage

Run 20 requests:
`nakrut "proxy-list.txt" "http://url.that.needs/views" 20`

Proxy-list file format:
`192.168.1.100\t8080\t\n  
192.168.1.101\t8081\t\n  
...`
